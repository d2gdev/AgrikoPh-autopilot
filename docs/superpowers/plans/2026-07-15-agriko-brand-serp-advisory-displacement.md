# Agriko Brand SERP 14-Day On-Domain SEO Sprint

**Goal:** Push FDA Advisory No. 2026-0489 outside the first 20 organic Google results for the exact Philippine query `agriko`, using only SEO work on `agrikoph.com` that implements or strengthens Agriko's active topical map.

**Execution boundary:** Every campaign change is made on `agrikoph.com`. There is no outreach, interview, external publishing, backlink campaign, profile work, marketplace work, social work, retailer work, or third-party execution.

## 1. Objective and Success Criteria

### Baseline

- Connected Windows Chrome observed the advisory at organic position 4 for `agriko` with `gl=ph`, `hl=en`, and `pws=0`.
- The same observation placed the Agriko homepage at position 1 and `/pages/about` at position 6.
- Serper returned an incomplete result window and did not reproduce the advisory. That absence is not valid displacement evidence.

### Success

- Interim milestone: browser-validated advisory position 11 or lower.
- Primary target: browser-validated advisory position 21 or lower, or absent from a complete first-20 result set.
- A selected Agriko asset receives displacement credit only when it is observed above the advisory.
- Moving the advisory one or two positions, publishing content, completing implementation, or receiving more impressions is not success by itself.

### Campaign limits

- Fourteen action days, beginning immediately.
- Measurement runs in parallel and continues automatically after day 14.
- No waiting period and no extended roadmap.
- No promise that multiple `agrikoph.com` URLs will all rank simultaneously; Google host diversity is an explicit constraint.

## 3. Current SERP Diagnosis

1. The advisory has exact brand wording in its title and URL and is hosted by an authoritative Philippine government domain.
2. The homepage already ranks first and should not be rewritten or retargeted.
3. About is the strongest change candidate because it already ranks at position 6 and has branded GSC visibility.
4. Find Agriko has strong branded GSC positioning but owns location intent; it must not be converted into a generic brand page.
5. The Christies Paglinawan and Farming Practices pages can strengthen Agriko's entity and provenance relationships while retaining their assigned intents.
6. The farming pillar, sustainable-rice spoke, and where-to-buy guide already have topical-map link requirements that are incomplete or weakly implemented.
7. Publishing new brand pages would duplicate existing ownership and create cannibalization risk.
8. On-domain SEO can strengthen the pages Google may choose, but host diversity means it cannot guarantee enough `agrikoph.com` listings to occupy the first 20.

## 10. Measurement Automation

### Task 0A: Correct provider coverage semantics

- Reuse `npm run seo:brand-serp:capture` and existing capture logic.
- A provider window shorter than 30 results records `provider_window_incomplete`, never `not_in_top_30`.
- Authentication, quota, rate-limit, and provider failures record failure states, never advisory absence.
- Preserve raw responses, provider, location, language, device, result depth, and `Asia/Manila` timestamps.

### Task 0B: Schedule four daily observations

- Add an authenticated cron route with `requireCronAuth(req)` first, `acquireJobLock`, matching release, and existing `JobRun` conventions.
- Run Philippines desktop/mobile and Cebu desktop/mobile observations daily.
- Upsert one read-only `RawSnapshot` per Manila calendar day without duplicate normalized observations.
- Record partial and total failures explicitly.
- Do not call Shopify or advertising mutation connectors.

### Task 0C: Browser validation exception workflow

- Reuse the connected Windows Chrome and Playwright extension where unattended execution is reliable.
- Preserve screenshots and normalized browser results.
- If unattended validation fails, alert once and create an explicit exception; do not make daily browser work the operator's responsibility.
- Browser evidence resolves provider disagreement.

### Task 0D: Reporting and alerts

- Produce a weekly report with advisory position, selected Agriko asset positions, indexation, GSC branded trends, releases, provider disagreement, and failures.
- Alert after repeated capture failures.
- Alert if the advisory returns to page one or the first 20 after crossing those thresholds.
- Link every observation to exact Content Proposal or Store Task release timestamps.

Implementation, tests, merge, deployment, scheduler installation, and production-health verification require their normal execution approvals. This planning document performs none of them.
